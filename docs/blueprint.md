# **App Name**: HanziCrafter

## Core Features:

- Radical Input: Allow users to input radicals for character creation via a user-friendly catalogue with drag and drop.
- Crafting Grid: Provide a 'crafting grid' interface where users can arrange radicals to form characters, similar to Minecraft's crafting table.
- Character Validation: Validate if the combination of radicals forms a valid Chinese character using a predefined dataset of radicals and character compositions.
- Generative Character Suggestions: AI tool suggest possible characters based on the radicals the user has placed in the crafting grid.
- Character Information: Display information about the created character, including its meaning, pronunciation (pinyin), and example usage.
- Character Saving: Allow users to save their crafted characters to a personal collection. Store locally or in a simple JSON file. No database MVP
- Challenge Mode: Present users with challenges to create specific characters using a limited set of radicals.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to evoke a sense of tradition and learning.
- Background color: Light, desaturated blue-gray (#ECEFF1) for a clean, non-distracting interface.
- Accent color: Gold (#FFC107) to highlight important elements and provide visual richness.
- Body and headline font: 'Literata', a transitional serif, providing a literary, vintage feel
- Use clean, minimalist icons to represent different radicals and actions, avoiding overly ornate designs.
- Crafting grid should be the central focus, with supporting information displayed in a clear, hierarchical manner. Follow the common practices of the Minecraft UI.
- Subtle animations when combining radicals and validating characters, providing satisfying feedback to the user.
